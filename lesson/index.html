Elements can be selected by their tag rather than their ID by simply excluding the '#' in your selector. For example, $("div") will select all div elements, regardless of ID.

The animation is achieved through a jQuery method called animate(). It allows you to animate changes to CSS properties of elements. The the method call will be animate({property: 'value'}).
